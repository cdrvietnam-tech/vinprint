# Runbook tiến hóa hằng tuần

Phân tích Search Console, GA4, click Zalo, lỗi, rejected memory và phản hồi của Đại ca. Tạo tối đa ba giả thuyết có bằng chứng. Mỗi giả thuyết phải ghi vấn đề, thay đổi, điểm trước/sau, chi phí, rủi ro, rollback và file ảnh hưởng.

Thay đổi hệ thống chỉ được đặt trên branch `agent-evolution/*` và mở PR có tiền tố `[Agent Evolution]`. Không merge, không push thẳng main và không thay đổi credentials. Gửi tóm tắt dễ đọc trong Codex để Đại ca phê duyệt.
